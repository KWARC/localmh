lmh module¶

Local Math Hub utility main parser.

Local MathHub XHTML conversion tool.

usage: lmh [-h] action ...
Sub-commands:
shell

launch a shell with everything set to run build commands. 

usage: lmh shell [-h] [shell]
Positional arguments:
shell shell to use
about

shows version and general information. 

usage: lmh about [-h]
find

find tool

usage: lmh find [-h] [--replace REPLACE] [--apply]
                matcher [repository [repository ...]]
Positional arguments:
matcher RegEx matcher on the path of the module
repository a list of repositories for which to show the status.
Options:
--replace Replace string
--apply=False Option specifying that files should be changed
status

shows the working tree status of repositories

usage: lmh status [-h] [--all] [repository [repository ...]]
Positional arguments:
repository a list of repositories for which to show the status.
Options:
--all=False, -a=False
 runs status on all repositories currently in lmh
log

shows recent commits in all repositories

usage: lmh log [-h] [--ordered] [--all] [repository [repository ...]]
Positional arguments:
repository a list of repositories for which to show the log.
Options:
--ordered=False, -o=False
 Orders log output by time (instead of by repository).
--all=False, -a=False
 runs log on all repositories currently in lmh
install

fetches a MathHub repository and its dependencies

usage: lmh install [-h] [repository [repository ...]]
Positional arguments:
repository a list of remote repositories to fetch locally. Should have form mygroup/myproject. No wildcards allowed.
setup

sets up local math hub and fetches external requirements

usage: lmh setup [-h] [--install | --update | --reinstall | --skip] [-f]
                 [--autocomplete] [--add-private-token ADD_PRIVATE_TOKEN]
                 [--latexml-source source@branch]
                 [--latexmls-source source@branch]
                 [--stex-source source@branch] [--mmt-source source@branch]
                 [--install-latexml | --update-latexml | --reinstall-latexml | --skip-latexml]
                 [--install-latexmls | --update-latexmls | --reinstall-latexmls | --skip-latexmls]
                 [--install-stex | --update-stex | --reinstall-stex | --skip-stex]
                 [--install-mmt | --update-mmt | --reinstall-mmt | --skip-mmt]
Options:
--install= Perform initial setup of dependencies. Default if no other arguments are given.
--update Update existing versions of dependencies. Implies –update-* arguments.
--reinstall Perform initial setup of dependencies.
--skip Skip everything which is not specified explicitly.
-f=False, --force=False
 Skip checking for lmh core requirements.
--autocomplete=False
 should install autocomplete for bash
--add-private-token
 add private token to use advanced MathHub functionality
--latexml-source=
 Get LaTeXML from the given source.
--latexmls-source=
 Get LaTeXMLs from the given source.
--stex-source= Get sTex from the given source.
--mmt-source= Get MMT from the given source.
--install-latexml=
 Install LaTexML.
--update-latexml
 Update LaTeXML.
--reinstall-latexml
 Reinstall LaTeXML.
--skip-latexml Leave LaTeXML untouched.
--install-latexmls=
 Install LaTexMLs.
--update-latexmls
 Update LaTeXMLs.
--reinstall-latexmls
 Reinstall LaTeXMLs.
--skip-latexmls
 Leave LaTeXMLs untouched.
--install-stex=
 Install sTeX.
--update-stex Update sTeX.
--reinstall-stex
 Reinstall sTeX.
--skip-stex Leave sTeX untouched.
--install-mmt= Install MMT.
--update-mmt Update MMT.
--reinstall-mmt
 Reinstall MMT.
--skip-mmt Leave MMT untouched.

gen

updates generated content

usage: lmh gen [-h] [-w number | -s] [-u | -f] [-n NICE | -H] [-v | -q]
               [--sms] [--omdoc] [--pdf] [--alltex] [--localpaths] [--list]
               [--pdf-add-begin-document] [--skip-implies]
               [-d RECURSION_DEPTH] [--all]
               [PATH_OR_REPOSITORY [PATH_OR_REPOSITORY ...]]
Positional arguments:
pathspec A list of paths or repositories to generate things in.
Options:
-w=8, --workers=8
 number of worker processes to use
-s, --single Use only a single process. Shortcut for -w 1
-u=False, --update=False
 Only generate files which have been changed. Experimental.
-f, --force Force to regenerate all files. DEFAULT.
-n=1, --nice=1 Assign the worker processes the given niceness.
-H, --high Generate files using the same priority as the main process.
-v=False, --verbose=False, --simulate=False
 Dump commands for generation to STDOUT instead of running them. Implies –quiet.
-q=False, --quiet=False
 Do not write log messages to STDOUT while generating files.
--sms=False generate sms files
--omdoc=False generate omdoc files, implies –sms, –alltex, –localpaths
--pdf=False generate pdf files, implies –sms, –alltex, –localpaths
--alltex=False Generate all.tex files
--localpaths=False
 Generate localpaths.tex files
--list=False dump all found modules to stdout. If enabled, –sms –omdoc and –pdf are ignored.
--pdf-add-begin-document=False
 add \begin{document} to LaTeX sources when generating pdfs. Backward compatibility for issue #82
--skip-implies=False
 Generate only what is requested explicitly. Might fail if some files are missing.
-d=-1, --recursion-depth=-1
 Recursion depth for paths and repositories.
--all=False, -a=False
 generates files for all repositories
