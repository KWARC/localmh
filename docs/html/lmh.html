lmh module¶

This is the entry point for the Local Math Hub utility.

usage: lmh [-h] action ...
Sub-commands:
status

shows the working tree status of repositories

usage: lmh status [-h] [repository [repository ...]]
Positional arguments:
repository a list of repositories for which to show the status.
install

fetches a MathHub repository and its dependencies

usage: lmh install [-h] [repository [repository ...]]
Positional arguments:
repository a list of remote repositories to fetch locally. Should have form mygroup/myproject. No wildcards allowed.
setup

sets up local math hub and fetches external requirements

usage: lmh setup [-h] [--autocomplete] [--add-private-token ADD_PRIVATE_TOKEN]
Options:
--autocomplete=False
 should install autocomplete for bash
--add-private-token
 add private token to use advanced MathHub functionality
xhtml

generate XHTML 

usage: lmh xhtml [-h] [repository [repository ...]]
Positional arguments:
repository a list of repositories for which to generate XHTML
init

initialize repository with MathHub repository structure

usage: lmh init [-h]
drain

send changes to mathhub

usage: lmh drain [-h] [--autocommit] [--message MESSAGE]
                 [repository [repository ...]]
Positional arguments:
repository a list of repositories for which to show the status.
Options:
--autocommit=False, -f=False
 should autocommit changes
--message=autocommit, -m=autocommit
 message to be used for commits
update

get repository and tool updates

usage: lmh update [-h] [repository [repository ...]]
Positional arguments:
repository a list of repositories for which to show the status.
gen

updates generated content

usage: lmh gen [-h] [-f] [-v] [--omdoc [OMDOC [OMDOC ...]]]
               [--pdf [PDF [PDF ...]]]
               [repository [repository ...]]
Positional arguments:
repository a list of repositories for which to show the status.
Options:
-f=False, --force=False
 force all regeneration
-v=False, --verbose=False
 verbose mode
--omdoc generate omdoc files
--pdf generate pdf files
clean

clean repositories of generated files

usage: lmh clean [-h] [repository [repository ...]]
Positional arguments:
repository a list of repositories for which to show the clean.
git

run git command on multiple repositories

usage: lmh git [-h] cmd [repository [repository ...]]
Positional arguments:
cmd a git command to be run.
repository a list of repositories for which to run the git command.
find

find tool

usage: lmh find [-h] [--replace REPLACE] [--apply]
                matcher [repository [repository ...]]
Positional arguments:
matcher RegEx matcher on the path of the module
repository a list of repositories for which to show the status.
Options:
--replace Replace string
--apply=False Option specifying that files should be changed
depcrawl

crawls current repository for dependencies

usage: lmh depcrawl [-h] [--apply]
Options:
--apply=False Option specifying that files should be changed
checkpaths

check paths for validity

usage: lmh checkpaths [-h] [--interactive] [repository [repository ...]]
Positional arguments:
repository a list of repositories for which to show the status.
Options:
--interactive=False
 Should check paths be interactive
repos

prints the group/repository of the current Math Hub repository

usage: lmh repos [-h]
root

prints the root directory of the Local Math Hub repository

usage: lmh root [-h]
sms

generates sms files

usage: lmh sms [-h] [repository [repository ...]]
Positional arguments:
repository a list of repositories.
mods

generates omdoc module files

usage: lmh mods [-h] [repository [repository ...]]
Positional arguments:
repository a list of repositories.
omdoc

generates omdoc for targets

usage: lmh omdoc [-h] [repository [repository ...]]
Positional arguments:
repository a list of repositories.
pdf

generates pdf for targets

usage: lmh pdf [-h] [repository [repository ...]]
Positional arguments:
repository a list of repositories.
modspdf

generates omdoc module files

usage: lmh modspdf [-h] [repository [repository ...]]
Positional arguments:
repository a list of repositories.
