Localmh

Welcome to the documentation of localmh.

Lmh, short for local mathhub, is an infrastructure for local (in a working tree) management of the content in http://MathHub.info.

As such it is mainly responsible for 2 different tasks:

  • Mathhub Repositories installation and management
  • External dependency management

It is still partially responsible for:

  • Generation content creation (to be taken over MMT)

Lmh is licensed under GPL v3, see License for details.

Installation

Installation should be done via localmh_docker.

In case a manual installation is needed, instructions can be found here.

Commands

Repository installation

  • lmh install - Installs Repositories

(External) dependencies and software updates

  • lmh setup - Sets up lmh and fetches external requirements
  • lmh selfupdate - Update lmh itself.

Generation process

Miscellaneous commands

  • lmh config - Configuration of localmh
  • lmh about - Shows version and general information
